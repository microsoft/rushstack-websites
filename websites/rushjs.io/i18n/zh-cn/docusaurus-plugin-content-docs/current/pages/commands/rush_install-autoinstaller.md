---
title: rush install-autoinstaller
---

```
用法：rush install-autoinstaller [-h] --name AUTOINSTALLER_NAME

使用该指令给一个项目安装依赖。

可选参数：
  -h, --help            展示帮助信息并退出
  --name AUTOINSTALLER_NAME
                        指定自动安装的包名，它必须是 common/autoinstallers
                        下的一个文件夹。
```
